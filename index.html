os-truenas-api-client signed feed

Signed FreeBSD pkg repository for the OPNsense plugin that vendors the TrueNAS WebSocket API client (midclt + truenas_api_client) so the os-acme-client truenas_ws deploy hook works on the firewall. Pin the key once — every install and upgrade is verified.

latest 0.1.0 ABI FreeBSD:14:amd64 pkg signature pubkey (RSA)

1Trust the feed

Pin the repository config and its signing key on the firewall (run as root).

# repo config + signing public key
fetch -o /usr/local/etc/pkg/repos/truenas-api-client.conf https://alphyriver.github.io/opnsense-truenas-api-client/truenas-api-client.conf
fetch -o /usr/local/etc/pkg/keys/truenas-api-client.pub  https://alphyriver.github.io/opnsense-truenas-api-client/pkg-repo.pub

2Install

Refresh the catalogue and install. Later upgrades are just pkg upgrade.

pkg update   # verifies the catalogue signature against the pinned key
pkg install os-truenas-api-client
